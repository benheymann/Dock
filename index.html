Dock

An Exhibition Format for Indexhibit

Dock is an Indexibit plugin that provides an additional presentation format for your exhibitions. A hiding dock, much like the dock in Apple OS X, is presented at the bottom of the exhibition screen, displaying thumbnails of the images in your exhibit. The remainder of the screen displays the currently selected image, automatically sized based upon your browser window size. Transitions between images and dock interactions are lightly animated for your viewing pleasure.

Examples

A live site running the Dock exhibition format can be viewed at tylerwilliamparker.com.

If you have an Indexhibit site that is using Dock, I'd be happy to link to you for example purposes. Please contact me and provide your site name and URL.

Installation

Installation is quick and easy.

  1. Optionally modify the default settings found in exhibit.dock.php.
  2. Place exhibit.dock.php in the "ndxz-studio/site/plugin/" folder.
  3. Upgrade the version of jquery.js that Indexhibit uses to 1.6.2 or newer. The easiest way to do this is to download the latest production version of jQuery and replace the file at "ndxz-studio/site/js/jquery.js" with it.
  4. From within the indexhibit administration interface, select "Settings" from the main "Exhibits" screen. Verify that "Advanced Mode" is set to "On". Click the "Update" button to save your changes.
  5. Edit an exhibit. Select "Dock" from the "Exhibition Format" drop-down menu. You're done!
